# Attribution Requirements

This repository and its demo projects are available under the MIT License with an additional attribution requirement.

## TL;DR

**Use it for anything you want, just keep the credit visible.**

## Allowed Uses

You may use this repository and its contents for:

- Personal projects
- Commercial projects
- Modification and redistribution
- Educational purposes
- Testing and development

## Attribution Requirements

If you use this repository or any of its demo projects, you must maintain attribution in the following locations:

### 1. Repository README

If you fork, clone, or redistribute this repository, the README must retain a visible link back to the original repository:

```
https://github.com/your-org/portscout-parody-stack
```

### 2. UI Footer Attribution

Each demo project includes a footer attribution block. This footer (or an equivalent credit page) must remain visible in any deployed or redistributed version.

**Required text:**

> Built with [PortScout Parody Stack](https://github.com/your-org/portscout-parody-stack) — created for testing [PortScout](https://github.com/your-org/portscout)

**Footer location:** `/branding/footer.html`

### Alternative Attribution

If the footer cannot be used, an equivalent credit must appear in:

- An "About" or "Credits" page
- The application settings
- A dedicated attribution section in the UI

The credit must be easily discoverable by users.

## Canonical Linkback Target

All attributions should link to:

```
https://github.com/your-org/portscout-parody-stack
```

## Questions?

If you're unsure about attribution requirements, feel free to open an issue in the repository. We're friendly and want you to use this work—just give credit where it's due!

---

_This attribution requirement is designed to be simple and reasonable. We appreciate your respect for the work that went into creating these demo projects._
