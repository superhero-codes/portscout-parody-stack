üöÄ Quick Start

From zero to twenty-six apps in three commands.

1. Clone the repo

git clone https://github.com/superhero-codes/portscout-parody-stack.git
cd portscout-parody-stack

2. Install everything (go make coffee ‚òï)

./scripts/install-all.sh
# Installs npm + pnpm deps across all projects
# Your patience will be tested.

3. Launch Chaos Mode

./scripts/start-all.sh # Unleash all apps
./scripts/stop-all.sh # When you've seen enough

4. Run a single project

# Projects are organized by category:
# projects/<category>/<name>

cd projects/streaming-and-chill/cornhub
npm install && npm start

cd projects/corporate-cringe/lotion
npm install && npm run dev

cd projects/chaos-gremlins/memeater
cargo run
