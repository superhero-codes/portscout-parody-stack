# Strife - Stripe Parody

A Stripe-style payment dashboard parody for PortScout demo purposes.

## What it Parodies

Stripe - the online payment processing platform.

## How to Start

```bash
npm install
npm run dev
```

## Port

- **HTTP**: 5305 (configured in `package.json` dev script)

## Notes

- Single-page mock dashboard with fake transactions
- Dark mode friendly
- No actual payment processing
