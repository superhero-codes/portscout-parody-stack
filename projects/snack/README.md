# Snack - Slack Parody

A Slack-style team communication parody for PortScout demo purposes.

## What it Parodies

Slack - the business communication platform for teams.

## How to Start

```bash
npm install
npm run dev
```

## Port

- **HTTP**: 5302 (configured in `vite.config.js`)

## Notes

- Single-page mock UI with channels and messages
- Dark mode friendly
- No actual messaging functionality
