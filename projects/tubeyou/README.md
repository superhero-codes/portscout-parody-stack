# TubeYou - YouTube Parody

A YouTube-style video platform parody for PortScout demo purposes.

## What it Parodies

YouTube - the video sharing platform.

## How to Start

```bash
npm install
npm start
```

Or for development:

```bash
npm run dev
```

## Port

- **HTTP**: 5303 (configured in `angular.json` serve options)

## Notes

- Single-page mock UI with video grid
- Dark mode friendly
- No actual video playback
