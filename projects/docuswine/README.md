# DocuSwine - DocuSign Parody

A DocuSign-style document signing parody for PortScout demo purposes.

## What it Parodies

DocuSign - the electronic signature platform.

## How to Start

```bash
npm install
npm run dev
```

## Port

- **HTTP**: 5309 (configured in `vite.config.ts`)

## Notes

- Single-page mock UI with document signing interface
- Dark mode friendly
- No actual document signing
