## ğŸš€ Quick Start

### Individual Projects

```bash

```

### Install Everything (Go Make Coffee)

```bash

```

> **Heads up:** This installs npm + pnpm dependencies across many projects. Your internet connection will be tested. Your patience will be tested more.

```bash

# When you've seen enough

```

> **Pro tip:** Running all projects at once is a great way to test your laptop's fan speed. Especially with MemEater and CPUStorm in the mix.
